# Release Notes

## v0.1.0 - Initial Public-Ready Package

Initial release candidate for AI Project Starter.

Highlights:

- Codex skill entrypoint with clear trigger description.
- Response modes for prompt-only, docs-only, plan-only, file creation, audit, repair, enterprise starter, vibe guardrails, and research-backed work.
- 60+ starter templates covering project docs, agent rules, domain packs, and ADRs.
- Source-backed reference map for Codex, Claude Code, Cursor, Continue, Copilot, Aider, Devin/Windsurf, AGENTS.md, PRD, architecture, ADR, and AI-agent safety sources.
- Scripted starter materializer with dry-run and no-overwrite defaults.
- Validation chain now includes a dry-run starter materializer smoke test through `npm run check`.
- Localized README pages are aligned with the `ai-project-starter` public identity and avoid stale English residue.
