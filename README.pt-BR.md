Pacote de skill para Codex que prepara contexto de projeto pronto para agentes, documentação inicial, guardrails de domínio e instruções de agente antes da implementação começar.

## Propósito

- transformar ideias brutas em contexto de produto, técnica, segurança e testes
- preparar `AGENTS.md`, briefs, especificações, planos e quality gates
- fornecer contexto consistente a ferramentas de AI coding em vez de prompts soltos
- proteger trabalho existente e criar arquivos apenas em modos explícitos
