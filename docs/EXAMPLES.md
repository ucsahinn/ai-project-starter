# Examples

## 1. Enterprise SaaS Cybersecurity Starter

```text
Use ai-project-starter in CREATE_FILES + ENTERPRISE_STARTER mode.

Project idea:
Tenant-based SaaS cybersecurity platform for vulnerability triage, RBAC, audit logs, secure evidence handling, AI-assisted remediation planning, and executive reporting.

Create the necessary project docs and agent instruction files before any coding starts.
```

## 2. Vibe Coding Guardrails Only

```text
Use VIBE_GUARDRAILS mode.

Project idea:
A fast prototype for a local-service booking web app.

Output only the docs and agent rules needed to prevent scope creep, duplicate systems, and unverified "done" claims.
```

## 3. Existing Project Context Audit

```text
Use AUDIT_CONTEXT mode.

Inspect README, AGENTS.md, architecture docs, testing docs, security docs, task docs, and any Cursor/Claude/Copilot rules. Lead with findings by severity.
```

## 4. Docs Only

```text
Use DOCS_ONLY mode.

Draft PROJECT_BRIEF.md, PRODUCT_REQUIREMENTS.md, TECHNICAL_SPEC.md, ARCHITECTURE.md, IMPLEMENTATION_PLAN.md, SECURITY.md, TESTING.md, and QUALITY_GATES.md for this project idea: [idea].
```
