Package de skill Codex qui prépare le contexte projet compatible avec les agents, la documentation de départ, les guardrails métier et les instructions d'agent avant l'implémentation.

## Objectif

- transformer une idée brute en contexte produit, technique, sécurité et test
- préparer `AGENTS.md`, briefs, spécifications, plans et quality gates
- donner un contexte cohérent aux outils d'AI coding au lieu de prompts isolés
- protéger le travail existant et créer des fichiers uniquement dans des modes explicites
